## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues; external PRs are also a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default canonical labels. See `docs/agents/triage-labels.md`.

### Domain docs

Use a single-context layout. See `docs/agents/domain.md`.

## Language

Use English for all future project contributions, including source code, code comments, tests, documentation, issue and pull request updates, and commit messages. User-facing localized strings remain governed by the project's i18n requirements.
